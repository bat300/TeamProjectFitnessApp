# TeamProjectFitnessApp

1) В консоли диспетчеров пакетов nuget, выбрав проект по умолчанию FitnessDietApp.Data, 
необходимо написать Update-Database, чтобы воспроизвести базу данных

2) В случае, если файл productsInfo.csv не найден, необходимо прописать абсолютный путь к нему

3) Для работы с API необходимо ввести ключ и id, которые можно найти по ссылке:
https://developer.edamam.com/edamam-docs-recipe-api
Для этого следует присвоить значения переменным AppKey и AppId в классе Repository, который находится в папке API Classes
